Practice Factor

Mar 11th, 2013

Factor is a programming language. I’m interested in this language.

Factor belong in a family tree of Stack-Oriented Programming Language. The tree include Forth and Joy.

Basic

2 + 5 = 7

( scratchpad ) 5 ! push one object to stack
--- Data stack:
5

( scratchpad ) 2
--- Data stack:
5
2

( scratchpad ) + ! 2 plus 7
--- Data stack:
7

We can reduce this program.

( scratchpad ) 2 5 +
--- Data stack:
7

Word

‘Word’ is like a function. For example +, /, dup, swap, map, and append are words in Factor.

( scratchpad ) 1 dup
--- Data stack:
1
1

( scratchpad ) -
--- Data stack:
0

( scratchpad ) 5
--- Data stack:
0
5

( scratchpad ) swap
--- Data stack:
5
0

( scratchpad ) clear ! clear stack

And we can define a word. Words can’t get arguments because words use the stack.

( scratchpad ) : my-increment ( x -- y )
    1 + ;

( scratchpad ) 4
--- Data stack:
4

( scratchpad ) my-increment
--- Data stack:
5

( scratchpad ) my-increment
--- Data stack:
6
